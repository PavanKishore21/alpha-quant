Workflow Guide
Portfolio
Select the stock basket

Choose 3 to 20 names from the universe table. Use search, sort, or the preset buttons to build the basket you want to test.

Setup
Set the run parameters

Choose the date range, rebalance frequency, max holdings, transaction cost, and chart scale before starting the run.

Defaults
Start with the default profile

A practical first run is a 1-year window, monthly rebalance, 5 holdings, and 10 bps cost. Change one setting at a time after you have a baseline.

Execution
Run the backtest

Click Run Backtest once the basket and parameters are ready. The live status in the header shows when market data is loading and when the run completes.

Results
Review the output below

Scroll down after the run to review the summary, return and risk metrics, charts, holdings, signal deck, and trade blotter before judging the result.
